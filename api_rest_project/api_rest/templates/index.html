Welcome to Api SmartList Compare with Django Python

Method Url Description
GET /categorias Lista de Categorias
GET /categorias/{id_categoria}/subcategorias Lista de Subcategorias por id_categoria
GET /subcategorias/{id_subcategoria}/tipos Lista de Tipos por id_subcategoria
GET /tipos/{id_tipo}/productos Lista de Productos por id_tipo
GET /usuarios Lista de Usuarios
GET /usuario/{nombre_usuario}/{contrasena} Objeto Usuario si coincide email y contraseña o {"usuario":null}
GET /supermercado/{id_supermercado}/productos Lista de Productos por id_supermercado
GET /supermercados Lista de Supermercados
GET /marcas Lista de Marcas
GET /marcas/{id_supermercado} Lista de Marcas por id_supermercado
GET /listacompra/{id_usuario}/productos Lista de la Compra de un Usuario por id_usuario
POST /usuario Add new usuario. <= JSON objeto usuario o {"msg":"Imcomplete Data..."} o {"msg": "User Already Exist..."}
POST listacompra/{id_usuario}/producto Add Producto a la Lista de la Compra. <= JSON objeto Lista Compra o {"msg":"Imcomplete Data..."} o {"msg": "Product already exists in the shopping list..."}
PUT /usuario/{email} Actualizar Usuario por su email (Unico de cada usuario porque los correos electronicos no se repiten)
DELETE /usuario/{email} Eliminar Usuario por su email (Unico de cada usuario porque los correos electronicos no se repiten)
DELETE /listacompra/{id_usuario}/producto/{id_producto} Eliminar Producto de la Lista de la Compra por id_usuario e id_producto
Nota La tabla Usuarios Tienen id AUTO_INCREMENT
